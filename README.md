# Instruksi Penggunaan

```cmd
$ git clone https://github.com/reighpuy/dcdreigh
$ cd dcdreigh
$ pip install -U discord.py
$ pip freeze > requirements.txt
```

  - Masukkan Token di *main_cogs.py*
  - Setelah dimasukkan langsung saja di Run dengan cara,
```cmd
$ python main_cogs.py
```
