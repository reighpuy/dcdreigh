# Usage Instruction

```cmd
$ git clone https://github.com/reighpuy/dcdreigh
$ cd dcdreigh
$ pip freeze > requirements.txt
```

  - Place your Bot Token at *bot.py*
  - Then Run with,
```cmd
$ python bot.py
```
