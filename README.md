# Instruksi Penggunaan

```cmd
$ git clone https://github.com/reighpuy/dcdreigh
$ cd dcdreigh
$ pip freeze > requirements.txt
```

  - Masukkan Token di *bot.py*
  - Setelah dimasukkan langsung saja di Run dengan cara,
```cmd
$ python bot.py
```
